# P4-AR-001 - AI Repair Proposal Engine

Status: DONE

## Objective

Introduce an isolated deterministic repair proposal layer that previews suggested fixes for supported blockers before any user-approved application exists.

Wave 1 only creates and displays proposals. Accept and reject record proposal decision state only. No CV mutation is allowed.

## Selected Execution

- Selected AI: Codex
- Selected model: GPT-5
- Selected reasoning level: 中
- Reason: This task adds a new isolated proposal layer, focused UI, and deterministic tests without modifying review, repair execution, export, prompt, persistence, or AI routing behavior.
- Escalation trigger: Browser/system validation may require approved localhost execution.

- Summary wording
- Work bullet wording
- External wording

- Duplicate wording

- Evidence mismatch
- Missing achievements
- Any repair requiring human judgement
