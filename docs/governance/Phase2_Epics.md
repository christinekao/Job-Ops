# Phase 2 Recommended Epics

Status: Recommendations only. No Phase 2 implementation is authorized by this file.

## Epic 1 - Contract Suite Integration

Add all focused Governance v1 regression commands to the canonical system test pipeline and improve contract-level failure reporting.

## Epic 2 - CV Evidence Traceability UX

Expose evidence coverage, generation-context drift, unsupported bullets, and repair provenance clearly in the active React UI.

## Epic 3 - Export and PDF Verification

Add browser/PDF rendering verification, text-layer checks, print-layout fixtures, and optional visual regression baselines.

## Epic 4 - Prompt and Contract Versioning

Formalize prompt/contract version compatibility, stale-version policy, and migration-safe reporting without making saved UX templates authoritative.

## Epic 5 - Governance Automation

Add lightweight checks for Task dependency/status consistency, Allowed/Forbidden file scope, governance links, ADR references, and completion-report completeness.

## Epic 6 - Legacy Archive Consolidation

Create explicit archive/migration documentation and remove legacy surfaces from active navigation or operational instructions without deleting historical artifacts.

## Epic 7 - CV Quality Calibration

Define role-family calibration, career-history compression rules, page-length variants, and outcome feedback loops using per-JD evidence rather than a canonical CV.
